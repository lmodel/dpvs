# DPV Extensions

Per-extension documentation for the W3C DPV 2.3 extension family. Each
extension is a standalone LinkML schema importing only the core semantic
group(s) it specialises; the pages below cover each extension's own
classes and slots, with core and cross-extension parents rendered as
plain text.

Per-jurisdiction `legal/` schemas and the `sector/` and `standards/`
trees are intentionally not rendered as separate documentation trees:
they re-slice the same core terms and would multiply build time without
adding distinct semantics. Browse their schemas directly under
[`src/dpv/schema/extensions/`](https://github.com/lmodel/dpv/tree/main/src/dpv/schema/extensions).
