# dpv_risk 

LinkML schema generated from the DPV 2.3 extension `risk`
(`risk-owl.ttl`), enriched with canonical SKOS metadata. Imports the
semantic-group schema(s) `dpv:schema/dpv_common`,
`dpv:schema/dpv_personal_data`, `dpv:schema/dpv_risk_notice` for the
abstract parent classes this extension specialises (instead of the full
umbrella `dpv:schema/dpv`).

URI: https://w3id.org/lmodel/dpv/risk
